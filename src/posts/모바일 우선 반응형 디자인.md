모바일 우선(Mobile First) 디자인은 작은 화면부터 시작하여 점진적으로 큰 화면에 맞춰 확장하는 접근법입니다.

## 모바일 우선의 장점

1. **성능**: 불필요한 리소스 로딩 방지
2. **사용자 경험**: 모바일 사용자에게 최적화된 경험 제공
3. **유지보수**: 작은 화면부터 시작하여 점진적 향상

## 미디어 쿼리 활용

```css
/* 모바일 우선 (기본 스타일) */
.container {
  padding: 1rem;
}

/* 태블릿 */
@media (min-width: 768px) {
  .container {
    padding: 2rem;
  }
}

/* 데스크톱 */
@media (min-width: 1024px) {
  .container {
    padding: 3rem;
    max-width: 1200px;
  }
}
```

## Tailwind CSS에서의 모바일 우선

```jsx
<div className="p-4 md:p-8 lg:p-12">
  {/* 모바일: p-4, 태블릿: p-8, 데스크톱: p-12 */}
</div>
```

## 터치 친화적 UI

- **터치 타겟 크기**: 최소 44x44px
- **간격**: 충분한 여백으로 실수 방지
- **제스처**: 스와이프, 핀치 등 자연스러운 제스처 지원

모바일 우선 접근법은 현대 웹 개발에서 필수적인 전략입니다.
