# 이미지 폴더 사용 가이드

## 이미지 등록 방법

### 방법 1: 게시글별 폴더 사용 (권장)

1. `public/images/{게시글-id}/` 폴더를 생성합니다.
   - 예: `public/images/네이버-부스트캠프-웹-모바일-10기-멤버십-12주차-회고/`

2. 이미지를 해당 폴더에 넣습니다.
   - 예: `public/images/네이버-부스트캠프-웹-모바일-10기-멤버십-12주차-회고/image1.png`

3. 마크다운에서 간단하게 참조합니다:
   ```markdown
   ![설명](image1.png)
   ```
   또는
   ```markdown
   ![설명](./images/image1.png)
   ```

### 방법 2: 공통 images 폴더 사용

1. `public/images/` 폴더에 직접 이미지를 넣습니다.
   - 예: `public/images/screenshot.png`

2. 마크다운에서 참조합니다:
   ```markdown
   ![설명](/images/screenshot.png)
   ```

## 게시글 ID 확인 방법

게시글의 ID는 제목에서 자동 생성됩니다:
- 공백 → 하이픈(-)으로 변환
- 예: "네이버 부스트캠프 웹・모바일 10기 멤버십 12주차 회고" → "네이버-부스트캠프-웹-모바일-10기-멤버십-12주차-회고"

## 이미지 스타일

이미지는 자동으로:
- 반응형으로 표시됩니다 (width: 100%)
- 둥근 모서리가 적용됩니다
- 그림자가 추가됩니다
- 지연 로딩(lazy loading)이 적용됩니다
- alt 텍스트가 있으면 이미지 아래에 캡션으로 표시됩니다
